1. 基本实例

    把一系列要使用的 .btn 按钮放入 .btn-group
    初始化：为标签加上.btn类，给父元素加上.btn-group类，再设置相应的参数。

  2. 按钮工具栏

    把一系列要使用的 .btn-group 元素放入 .btn-toolbar
    初始化：为标签加上.btn-group类，给父元素加上.btn-toolbar类，再设置相应的参数。

  3. 垂直排列

    使用 .btn-group-vertical 使按钮垂直排列显示
    初始化：为标签加上.btn类，给父元素加上.btn-group-vertical类，再设置相应的参数。

  4. 两端对齐排列

    添加 .btn-group-justify类， 让按钮组里的按钮平均分布，填满容器宽度
    注意： 只适用 a 元素，button 不能应用 display: table-cell 样式
    使用 flexbox 实现，只兼容 IE 10+ 及其他现代浏览器
    初始化：为标签加上.btn类，给父元素加上.btn-group-justify类，再设置相应的参数。
