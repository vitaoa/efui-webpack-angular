1. 页面编码规范

    1. css文件
      应放到body的前面head里面。而且要使用rel属性。 
      <link href="assets/css/index.css" rel="stylesheet">
    2. 统一使用 UTF-8 编码,用@charset "utf-8"指定页面编码。
    3. 全局字体设置：
      • windows 7系统：微软雅黑, Arial
      • windows XP及以下：新宋体，宋体，Arial
      • MAC默认字体：Helvetica Neue和Helvetica Hiragino Sans GB
      • font-family:"Microsoft Yahei","Hiragino Sans GB","Helvetica Neue",Helvetica,tahoma,arial,Verdana,sans-serif,"WenQuanYi Micro Hei","\5B8B\4F53";
    4. 中文字体使用编码转换

  2. 书写规范

    1. 书写建议：
      • 选择符、属性、值均用小写（格式的颜色值除外）,在xhtml标准中规定了所有标签、属性和值都小写，CSS也是如此
      • 属性写在一行内，属性之间、属性名和值之间以及属性与“{}”之间应减少空格，最后一个值也以分号结尾 
        例如：
.class{width:200px;height:100px;}
      • 省略值为0时的单位 
        .m-box{margin:0 10px;background-position:  50% 0;}
    2. 书写顺序：
      1. 按照元素模型由外及内，由整体到细节书写，大致分为五组：(显示属性->自身属性->文本属性和其他修饰)
        • 位置：position,left,right,float
        • 盒模型属性：display,margin,padding,width,height
        • 边框与背景：border,background
        • 段落与文本：line-height,text-indent,font,color,text-decoration,...
        • 其他属性：overflow,cursor,visibility,...
        显示属性 自身属性 文本属性和其他修饰
          display width font
          visibility height text-align
          position margin text-decoration
          float padding vertical-align
          clear border white-space
          list-style overflow color
          top min-width background
        针对特殊浏览器的属性，应写在标准属性之前，遵循私有在前，标准在后原则 
        例如：
-webkit-box-shadow:;
-moz-box-shadow:;
box-shaow:;
      2. 伪类书写顺序：
        • :link :visited :hover :active书写顺序 L-V-H-A，速记：LoVe（喜欢）HAte（讨厌）

  3. 注释格式

    1. 对选择器的注释统一写在被注释对象的上一行
      /* 单行注释文字 */
.m-list li em{color:#666;}
    2. 对属性及值的注释写于分号后
      .m-list li{height:20px;line-height:  20px;/* 这里是对line-height的一个注释 */overflow:hidden;}
    3. 块状注释，块状注释保持统一的缩进对齐
      /* 块状注释文字 
 * 块状注释文字 
 * 块状注释文字 
 */
.m-list{width:500px;}

  4. 原则上不允许使用Hack

    很多不兼容问题可以通过改变方法和思路来解决，并非一定需要Hack，根据经验你完全可以绕过某些兼容问题。 一种合理的结构和合理的样式，是极少会碰到兼容问题的。 由于浏览器自身缺陷，我们无法避开的时候，可以允许使用适当的Hack。
    统一Hack方法：
    • 统一使用“*”和“_”分别对IE7和6进行Hack 
      /* IE7会显示灰色#888，IE6会显示白色#fff，其他浏览器显示黑色#000 */
.m-list{color:#000;*color:  #888;_color:#fff;}

  5. 选择器优先级

    • a = 行内样式style。
    • b = ID选择器的数量。
    • c = 类、伪类和属性选择器的数量。
    • d = 类型选择器和伪元素选择器的数量。
